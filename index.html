Requisitos

Windows (Testeado en Windows 11 24H2)

  • +8 gigabytes de RAM.

    • Para poder ver la RAM que tienes disponible en tu dispositivo, presiona Win + X, va a aparecer una lista con opciones, presiona la opcion "Sistema". Deberia abrirse una ventana como esta:

    Debes fijarte en la parte donde dice "RAM Instalada", como vemos en la imagen, mi equipo tiene 32 gigabytes de RAM, por lo tanto mi equipo cumple este requisito.

  • Java 11 instalado.

    • En caso de que no tengas instalado Java 11, tengas otra version o no estes seguro de que lo tienes, dirigete a el primer paso en las instrucciones de inicio de Windows.

  • Token de Ngrok cargado.

    • Para cargar el token ve a el primer paso en las instrucciones de inicio de Windows.

  • Editor de texto compatible con los siguientes tipos de archivo: .json, .xml, .propierties. (El bloc de notas es compatible)

Linux (Testeado en Ubuntu Desktop 24.04.1)

  • +8 gigabytes de RAM.

    • Para verificar cuanta RAM tienes en tu dispositivo, abre el Monitor del Sistema, deberia aparecerte algo asi:

    Hay que fijarnos abajo de la segunda grafica, donde dice Memoria, podras ver que dice A GB (B%) de C GB. A es la cantidad de RAM usada en gigabytes, B es el porcentaje de RAM usada y C es la RAM que posee tu equipo, mi equipo tiene 12.2 gigabytes, por lo que cumple el requisito.

  • Java 11 instalado.

    • En caso de no estar instalado, estar instalado en otra version o no estar seguro, ve al primer paso de Instrucciones de Inicio para Linux.

  • Ngrok para la terminal descargado.

    • En caso de no estar instalado, estar instalado en otra version o no estar seguro, ve al primer paso de Instrucciones de Inicio para Linux.

  • Token de Ngrok cargado.

    • En caso de no estar cargado, revisa el primer paso de Instrucciones de Inicio en Linux.

  • Editor de texto compatible con los siguientes tipos de archivo: .json, .xml, .propierties. (Editor de texto y Notepad++ son compatibles)

Instrucciones de inicio

Windows

  1. Suponiendo que el USB ya está conectado al ordenador, el siguiente paso es verificar los requisitos fundamentales: Java 11 y Ngrok.

    Java 11

    Para comprobar Java en cualquier versión de Windows debemos abrir la terminal y ejecutar el siguiente comando:

    java -version

    En caso de que no tengas Java te debe aparecer algo así:

    'java' no se reconoce como un comando interno o externo, programa o archivo por lotes ejecutable.

    En caso de que tengas alguna versión de Java te debe aparecer esto:

    java version "17.0.2" 2022-01-18 LTS
Java(TM) SE Runtime Environment (build 17.0.2+8)
Java HotSpot(TM) 64-Bit Server VM (build 17.0.2+8, mixed mode)

    Para verificar que tengas la versión de Java, debes verificar el inicio del comando, donde dice java version "17.0.2", el número 17 debería ser 11 y los otros dos números pueden ser cualquiera.

    En caso de no tener Java 11 o tener otra versión

    Ir a la carpeta Programas en el USB, ir a Windows y ejecutar el archivo por lotes: Java.msi. (Elegir 64 o 32 bits según tu equipo).

    Para ver de cuántos bits es tu equipo, haz clic derecho en el botón Windows y selecciona “Sistema”. En la sección “Especificaciones del dispositivo”, verás el “Tipo de sistema”. Si dice “PC basado en x64”, es de 64 bits. Si dice “PC basado en x86”, es de 32 bits.

    Al terminar la instalación, ya debería estar instalado Java 11, superponiendo a la versión anterior en caso de tener otra versión de Java o no tener Java.

    Ngrok

    En Windows, Ngrok es muy fácil de usar, no necesitas instalar nada, en la carpeta ServerITC hay 2 archivos; Ngrok.exe y Ngrok_32bits.exe. Debes elegir según tu equipo:

    • Si tu equipo es de 64 bits, selecciona el archivo Ngrok.exe.
    • Si tu equipo es de 32 bits, renombra el archivo Ngrok.exe a Ngrok_64bits.exe y renombra el archivo Ngrok_32bits.exe a Ngrok.exe.

    Ya teniendo abierto Ngrok.exe, si todavía no tienes ningún token de Ngrok cargado en tu computador, escribe:

    ngrok config add-authtoken 2qwt0gapRolUawhPzQ8yM9TJCeW_3PMAAF7dSzeDNdMViJxCP

    Si ya tenemos el token cargado podemos cerrar Ngrok.exe.

  2. Configurar la RAM disponible para el servidor. Para esto, abre el archivo startin_windows.bat con un editor de texto y deberias ver el siguiente código: 
    TITLE ServerITC
:loop
 java -Xmx4G -Xms4G -jar forge-1.16.5-36.2.34.jar nogui
goto :loop
pause

    Para configurar la RAM, modifica la línea 3, cambiando 4G por la cantidad deseada de gigabytes en ambos valores.

    Ejemplo:

    TITLE ServerITC
:loop
 java -Xmx999G -Xms999G -jar forge-1.16.5-36.2.34.jar nogui
goto :loop
pause
  3. Inicia el servidor ejecutando con doble clic el archivo startin_windows.bat y espera a que se inicie. Para comprobar si ya inicio, presiona dos veces la tecla Enter. Si recibes una respuesta como: 
    Unknown or incomplete command, see below for error at position 1: <--[HERE]

    El servidor ya está iniciado.

    Para acceder al servidor, usa Ngrok:

    Ejecuta ngrok.exe y escribe el siguiente comando:

    ngrok tcp 25565

    Deberia aparecerte una tabla asi:

    En la tabla podemos ver 2 columnas, en la columna de la izquierda busca "Forwarding", en esa misma fila, columna derecha, hay una url: tcp://0.tcp.sa.ngrok.io:18781 la IP del servidor que debes usar y compartir es 0.tcp.sa.ngrok.io:18781 excluyendo el tcp://. Sin embargo el numero que aparece ahi; 18781, te va a aparecer diferente a vos, esto es porque por cada vez que se abra el server con Ngrok, ese numero cambia, por eso debes compartir la IP cada vez que abras el server. Por ejemplo si yo abro de nuevo Ngrok, me aparece el siguiente numero:

    Podemos ver que esta vez me toco el numero 17634, por lo que la IP que debemos usar para el servidor hasta que lo cerremos y lo abramos de nuevo.

    Ahora que ya tenemos abierto Ngrok, ya podemos unirnos al servidor. Abrimos Minecraft con Forge 1.16.5 con los mods instalados y vamos a la seccion de Multijugador. Apretamos Agregar servidor y le ponemos un nombre al servidor. Donde dice IP del servidor ponemos la IP que conseguimos en Ngrok.

    Apretamos hecho y ya podemos entrar al servidor.

Linux

  1. Suponiendo que el USB ya está conectado al ordenador, el siguiente paso es verificar los requisitos fundamentales: Java 11 y Ngrok.

    Java 11

    Para comprobar Java en cualquier versión de Linux debemos abrir la terminal y ejecutar el siguiente comando:

    java -version

    En caso de que no tengas Java te debe aparecer algo así:

    Command 'java' not found

    En caso de que tengas alguna versión de Java te debe aparecer esto:

    openjdk version "17.0.4" 2023-01-17
OpenJDK Runtime Environment (build 17.0.4+8)
OpenJDK 64-Bit Server VM (build 17.0.4+8, mixed mode)

    Para verificar que tengas la versión de Java, debes verificar el inicio del comando, donde dice openjdk version "17.0.4", el número 17 debería ser 11 y los otros dos números pueden ser cualquiera.

    En caso de tener otra versión

    Hay que eliminar todas las versiones de Java descargadas, de otra forma las otras versiones tendrán mas prioridad. Para esto ejecutaremos:

    sudo apt-get remove --purge openjdk-*
    sudo apt-get autoremove
    sudo apt-get purge openjdk-*

    Ahora para terminar de asegurarnos vamos a ver todas las versiones de Java descargadas.

    sudo update-alternatives --config java

    Te va a aparecer una tabla como esta, presiona Ctrl + C para salir del editor de prioridades (la tabla) y todas esas rutas se quedaran en el historial de la terminal. Ahora para cada ruta vas a ejecutar este comando y donde dice /ruta/java reemplaza con la ruta correspondiente. Repite para cada ruta que haya aparecido en la tabla.

    sudo rm -rf /ruta/java

    Ejemplo con las rutas que aparecen en mi tabla (en verdad yo no deberia porque la version que aparece ahi es Java 11)

    sudo rm -rf /usr/lib/jvm/bellsoft-java11-full-amd64/bin/java

    En caso de no tener Java 11

    Ir a la carpeta Programas en el USB, ir a Linux, abrir la terminal en esa ubicación y ejecutar el siguiente comando.

    sudo dpkg -i Linux.deb

    Al terminar, ya debería estar instalado Java 11.

    Ngrok

    En Linux, para instalar Ngrok, debes abrir una nueva terminal y ejecutar el siguiente comando:

    curl -sSL https://ngrok-agent.s3.amazonaws.com/ngrok.asc \
    | sudo tee /etc/apt/trusted.gpg.d/ngrok.asc >/dev/null \
    && echo "deb https://ngrok-agent.s3.amazonaws.com buster main" \
    | sudo tee /etc/apt/sources.list.d/ngrok.list \
    && sudo apt update \
    && sudo apt install ngrok

    Ya teniendo instalado Ngrok, si todavía no tienes ningún token de Ngrok cargado en tu computador, escribe:

    ngrok config add-authtoken 2qwt0gapRolUawhPzQ8yM9TJCeW_3PMAAF7dSzeDNdMViJxCP

    Si ya tenemos el token cargado podemos cerrar la terminal.

  2. Configurar la RAM disponible para el servidor. Para esto, abre el archivo startin_linux.sh con un editor de texto y busca el siguiente código: 
    echo "Iniciando el ServerITC..."
while true 
do
    java -Xmx4G -Xms4G -jar forge-1.16.5-36.2.34.jar nogui
    echo "El servidor se detuvo. Reiniciando..."
done

    Para configurar la RAM, modifica la línea 4, cambiando 4G por la cantidad deseada de gigabytes en ambos valores.

    Ejemplo:

    echo "Iniciando el ServerITC..."
while true 
do
    java -Xmx999G -Xms999G -jar forge-1.16.5-36.2.34.jar nogui
    echo "El servidor se detuvo. Reiniciando..."
done
  3. Abre una terminal en la carpeta ServerITC y ejecuta el siguiente comando: 
    bash startin_linux.sh

    Espera a que se inicie el servidor, para verificar si ya termino de iniciar, escribe bsk y presiona Enter. Si la terminal no da ninguna respuesta, el server sigue iniciando, en cambio si muestra el siguiente texto, significa que ya inicio el server.

    Unknown or incomplete command, see below for error at position 1: <--[HERE]

    El servidor ya está iniciado. Ahora vamos a abrir Ngrok para poder unirnos

    Abre una terminal en la carpeta ServerITC y ejecuta el siguiente comando:

    ngrok tcp 25565

    Deberia aparecerte una tabla asi:

    En la tabla podemos ver 2 columnas, en la columna de la izquierda busca "Forwarding", en esa misma fila, columna derecha, hay una url: tcp://0.tcp.sa.ngrok.io:18781 la IP del servidor que debes usar y compartir es 0.tcp.sa.ngrok.io:18781 excluyendo el tcp://. Sin embargo el numero que aparece ahi; 18781, te va a aparecer diferente a vos, esto es porque por cada vez que se abra el server con Ngrok, ese numero cambia, por eso debes compartir la IP cada vez que abras el server.     Podemos ver que esta vez me toco el numero 17634, por lo que la IP que debemos usar para el servidor hasta que lo cerremos y lo abramos de nuevo.

    Ahora que ya tenemos abierto Ngrok, ya podemos unirnos al servidor. Abrimos Minecraft con Forge 1.16.5 con los mods instalados y vamos a la seccion de Multijugador. Apretamos Agregar servidor y le ponemos un nombre al servidor. Donde dice IP del servidor ponemos la IP que conseguimos en Ngrok.
